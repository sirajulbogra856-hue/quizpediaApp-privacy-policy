## 1. Information We Collect

### 1.1 Automatically Collected Data

When you use the App, we may automatically collect:

| Data Type | Examples | Source |
|----------|----------|--------|
| Device Information | Device model, OS version, IP address, app version | Firebase, Google Analytics |
| Usage Data | Screens visited, time spent in app, features used | Firebase Analytics |
| Crash Information | Crash logs, error identifiers | Firebase Crashlytics |
| Advertising Data | Google Advertising ID (AAID), ad interactions | Google AdMob |

We **do not collect personally identifiable information (PII)** such as your name, email, phone number, or address.

### 1.2 Ad Data (Google AdMob)

We use **Google AdMob** to show advertisements.  
AdMob may collect and use data such as:

- Advertising Identifier (AAID)
- IP address
- Device identifiers
- Interactions with ads

AdMob may use this data to show personalized ads unless you opt out.

Learn more:  
https://policies.google.com/technologies/ads

### 1.3 Firebase Services

We use Firebase to improve app quality.

| Firebase Service | Purpose |
|------------------|---------|
| Firestore | Stores quiz questions |
| Analytics | Usage statistics & app performance |
| Crashlytics | Crash detection & debugging |

Firebase may receive:
- device & usage information
- crash reports
- general analytics data

Firebase Privacy Information:  
https://firebase.google.com/support/privacy
